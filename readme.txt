Detects if a user is using adBlock, adBlock Plus or any other software that might be disabling ads on your website and prompts them to disable it.

== Description ==

Stop losing money on your google adsense and other ad network accounts by using adBlock Alerter. adBlock Alerter detects if a user on you blog/website is using; adBlock, adBlock Pro, adBlock Plus or any type of ad blocking plugins/extensions and displays a message or image to disable it/add your website to their whitelist before viewing the content. The message you display is totally customizable and can even be an image.

== Installation ==

How to install adBlock Alerter and get it working.

1. Install the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On your left sidebar click 'adBlock Alerter' and set up your page
